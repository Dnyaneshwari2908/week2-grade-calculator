# Student Grade Calculator

## Project Description
This project calculates student grades using Python.

It uses:
- Conditional statements
- Lists
- Loops
- Functions
- Error handling
- File handling

---

## Features
- Calculates average marks
- Assigns grades
- Provides comments
- Displays class statistics
- Searches student records
- Saves results to file
- Handles invalid input

---

## Technologies Used
- Python

---

## How to Run

1. Open terminal
2. Navigate to project folder
3. Run: python grade_calculator.py
